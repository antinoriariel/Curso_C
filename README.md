
# Curso Completo de C — Desde Cero hasta Nivel Avanzado

**Autor:** Kernel  
**Nivel:** Principiante → Avanzado  
**Enfoque:** 100% práctico, académico y actualizado (C17/C23)

## Descripción General

Este curso cubre el lenguaje de programación C en su totalidad, partiendo desde los fundamentos básicos hasta tópicos avanzados de sistemas, memoria, concurrencia, redes, compiladores, kernels, HPC, criptografía, machine learning, robótica y más. Con **40 capítulos** y más de **200 subcapítulos**, es el recurso más completo en habla hispana para aprender C. Cada capítulo está diseñado para ser autocontenido, con ejercicios prácticos, ejemplos compilables y documentación detallada.

## Objetivos

- Dominar la sintaxis y semántica del lenguaje C
- Comprender la gestión manual de memoria y el modelo de hardware subyacente
- Implementar estructuras de datos y algoritmos clásicos
- Desarrollar aplicaciones concurrentes, en red y de bajo nivel
- Escribir código robusto, eficiente y libre de comportamientos indefinidos
- Prepararse para sistemas embebidos, kernels, compiladores, HPC, criptografía, machine learning y robótica

## Requisitos Previos

- Conocimientos básicos de computación (archivos, terminal)
- Un compilador de C (GCC, Clang, MSVC)
- Un editor de texto o IDE (VS Code, CLion, Vim)
- Ganas de aprender a fondo

## Metodología

- **Teoría concisa** seguida de **código compilable**
- **Ejercicios progresivos** con soluciones
- **Proyectos acumulativos** que integran múltiples conceptos
- **Peer review** y análisis de código mediante herramientas estáticas (Valgrind, AddressSanitizer, Clang-Tidy)
- **Exámenes prácticos** al final de cada módulo

## Cómo Usar Este Repositorio

```bash

cd Curso_C

# Compilar un ejemplo del capítulo N

./programa
```
